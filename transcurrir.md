# Transcurrir

## 📖 基础信息

| 项目 | 内容 |
|------|------|
| **单词** | **Transcurrir** |
| **词性** | tr. 及物动词 |
| **核心释义** | intr.       （时间）消逝，度过 |

## 🧠 记忆辅助

- **联想记忆**：词根记忆：*Transcurrir* → 词干 *transcurrir* + 动词后缀 *-ar/-er/-ir*，表示动作或状态。

## ✏️ 核心用法

**及物动词**：直接接宾语，表示'intr.       （时间）消逝'
- 结构：Transcurrir + 名词/代词

1. El tiempo transcurre muy de prisa. 时光飞逝。
