# por ejemplo

## 📖 基础信息

| 项目 | 内容 |
|------|------|
| **表达** | **por ejemplo** |
| 音标 | [poɾ emˈplo.ɣo] |
| 词性 | 副词短语 (locución adverbial) |
| 核心释义 | 例如 |

---

## 🧠 记忆辅助

**联想记忆：**
- por（通过）+ ejemplo（例子）→ "通过例子" → 例如
- 联想：for example → 例如

**词源补充：**
- ejemplo 来自拉丁语 exemplum（例子）
- 字面意思是"通过例子"，引申为"例如"

---

## ✏️ 核心用法

### 1. 表示举例
用于举例说明前述内容。

### 2. 表示补充
用于补充具体例子。

### 3. 通用用法
可用于口语和书面语。

### 4. 固定搭配
- por ejemplo, como：例如，比如

---

## 📝 例句模块

### 用法1：表示举例
**Por ejemplo, el proyecto fue un éxito.**
例如，项目是成功的。

**Por ejemplo, los resultados son positivos.**
例如，结果是积极的。

### 用法2：表示补充
**Por ejemplo, hay otros factores a considerar.**
例如，还有其他因素需要考虑。

**Por ejemplo, la situación es compleja.**
例如，情况很复杂。

### 用法3：通用用法
**Por ejemplo, la normativa es estricta.**
例如，法规很严格。

**Por ejemplo, los requisitos son mínimos.**
例如，要求是最低的。

### 用法4：固定搭配
**Por ejemplo, como mencioné antes, se necesita más inversión.**
例如，正如我之前提到的，需要更多投资。

**Por ejemplo, como ya he dicho, el proyecto fue un éxito.**
例如，正如我已经说过的，项目是成功的。
