# Circular

## 📖 基础信息

| 项目 | 内容 |
|------|------|
| **单词** | **Circular** |
| **音标** | [θiɾ.ku.ˈlaɾ] |
| **词性** | adj. / f. (形容词/阴性名词) |
| **核心释义** | 圆形的；通知，通告 |

---

## 🧠 记忆辅助

**联想记忆：**  
"Circular" = "círcul"（圆圈）+ "ar"（形容词后缀）。想象"绕圈走的东西"——圆形的桌子、循环的通知（circular oficial）。

**词源补充：**  
来自拉丁语 *circularis*（圆的），来自 *circus*（圆环）。作名词阴性时指"通知/通告"——因为通告是发给"每个人循环"看的。

---

## ✏️ 核心用法

### 1. 形容词：圆形的/环形的
修饰形状，描述事物的几何特征。

### 2. 名词（阴性）：通知/通告
官方文件，发给相关人员的循环通知。

### 3. 动词 circular 的变位形式
需注意词性：形容词 circular，重音在 o；动词 circular，重音在 u。

## 📝 例句（6句）

1. **La mesa circular es ideal para espacios pequeños.**  
   圆桌适合小空间。

2. **Recibimos una circular del ministerios sobre los nuevos criterios.**  
   我们收到了部里关于新标准的通知。

3. **El movimiento circular del planeta se llama rotación.**  
   行星的圆周运动叫做自转。

4. **La staircase circular es el encanto principal de este edificio.**  
   旋转楼梯是这座建筑的主要魅力。

5. **La circular informaba a todos los empleados sobre el cambio de horario.**  
   通知告知所有员工时间变更。

6. **El argumento del libro sigue una estructura circular.**  
   这本书的论证呈圆形结构。
