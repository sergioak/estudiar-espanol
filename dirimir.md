# dirimir

## 1. 📖 基础信息

| 项目 | 内容 |
|------|------|
| **单词** | dirimir |
| **音标** | /diɾiˈmiɾ/ |
| **词性** | 及物动词 (verbo transitivo) |
| **核心释义** | 解决（争端）、调解、裁决 |

---

## 2. 🧠 记忆辅助

**联想记忆：**
联想 "dirigir"（指挥、引导）——要**解决（dirimir）**争端，就需要有人站出来"指挥方向"。也可以拆分为 "di-rimir"，"rimir" 联想到 "resolve"（解决），两个词都以 "r" 开头，且含义相通。

**词源补充：**
源自拉丁语 *dirimere*，由 *dis-*（分开）+ *emere*（拿取）构成，字面意思为"拿开、分离"，引申为"将争端分开/化解"。在教会法（derecho canónico）中有专门用语 **causa dirimente**（使婚姻无效的原因）。

---

## 3. ✏️ 核心用法

| 用法模式 | 说明 |
|----------|------|
| **dirimir un conflicto/una disputa** | 解决冲突/争端。最核心的搭配，正式语体。 |
| **dirimir una controversia** | 解决争议。法律、学术和外交语境。 |
| **dirimir las diferencias** | 化解分歧。政治和商业谈判中常见。 |
| **dirimir la cuestión** | 解决问题/裁决事项。正式书面语。 |

---

## 4. 📝 例句模块

1. **El tribunal internacional fue convocado para dirimir el conflicto territorial entre ambos países.**
   国际法庭被召集来裁决两国之间的领土争端。

2. **Las partes acordaron dirimir sus diferencias mediante la mediación.**
   双方同意通过调解来化解分歧。

3. **Es necesario dirimir esta controversia antes de avanzar en las negociaciones.**
   在推进谈判之前，有必要先解决这一争议。

4. **El árbitro dirimió la disputa laboral con una sentencia equitativa.**
   仲裁人以公正的裁决解决了劳动纠纷。

5. **La asamblea extraordinaria se reunió para dirimir las cuestiones pendientes.**
   特别大会召开会议以解决悬而未决的问题。

6. **Los líderes políticos buscaron dirimir la crisis sin recurrir a la violencia.**
   政治领导人寻求在不诉诸暴力的情况下解决危机。

7. **El juez dirimió la cuestión a favor de la parte demandante.**
   法官裁决此事项，支持原告一方。

8. **No hay foro adecuado para dirimir estas desavenencias entre hermanos.**
   没有合适的平台来解决兄弟姐妹之间的这些分歧。

**答案与解析：**

| 题号 | 答案 | 解析 |
|------|------|------|
| 1 | a) dirimir | 不定式原形，用于 para + inf. 结构表示目的。 |
| 2 | b) legal, diplomático y formal | dirimir 是高雅语体词汇，常用于法律、外交和正式场合。 |
| 3 | c) crear | crear（创造）不是 dirimir 的近义词。resolver、zanjar、solucionar 均为近义。 |
| 4 | a) dirimir | 及物动词直接接宾语 necesitaban dirimir sus diferencias。 |
| 5 | b) separar / apartar | 拉丁语 dirimere 由 dis-（分开）+ emere（拿取）构成，意为"分开、拿开"。 |
| 6 | a) dirimir | para + 不定式原形，表示目的。 |
| 7 | d) a y c son correctas | 两句都正确使用了 dirimir：a 为正式语境，c 也可用于较正式的邻里纠纷。 |
| 8 | a) dirimente | 教会法术语 causa dirimente（使婚姻无效的原因）。 |
| 9 | b) "dirimir" tiene un registro más formal | dirimir 比 resolver 更正式，专用于法律、外交等高雅语境。 |
| 10 | a) dirimir | lograr + 不定式原形，意为"成功解决了..."。 |
