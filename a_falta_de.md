# a falta de

## 📖 基础信息

| 项目 | 内容 |
|------|------|
| **表达** | **a falta de** |
| 音标 | [a ˈfal.ta de] |
| 词性 | 介词短语 (locución prepositiva) |
| 核心释义 | 由于缺乏、因为没有 |

---

## 🧠 记忆辅助

**联想记忆：**
- falta（缺乏）→ a falta de → 由于缺乏
- 联想：fault（错误）→ 缺乏 → 由于没有

**词源补充：**
- falta 来自拉丁语 fallita（失败）
- 字面意思是"在缺乏的情况下"，引申为由于缺乏

---

## ✏️ 核心用法

### 1. 表示原因
用于引出由于缺乏某物而产生的结果。

### 2. 表示替代
用于表示在没有某物的情况下使用替代品。

### 3. 正式用法
主要用于书面语和正式场合。

### 4. 固定搭配
- a falta de pruebas：由于缺乏证据
- a falta de recursos：由于缺乏资源

---

## 📝 例句模块

### 用法1：表示原因
**A falta de pruebas, fue absuelto.**
由于缺乏证据，他被无罪释放。

**A falta de presupuesto, el proyecto se canceló.**
由于缺乏预算，项目被取消。

### 用法2：表示替代
**A falta de azúcar, usa miel.**
没有糖，就用蜂蜜。

**A falta de pan, tortillas.**
没有面包，就吃玉米饼。

### 用法3：正式用法
**A falta de alternativas, se decidió continuar.**
由于缺乏替代方案，决定继续。

**A falta de información, no se puede decidir.**
由于缺乏信息，无法决定。

### 用法4：固定搭配
**A falta de pruebas, el caso se archivó.**
由于缺乏证据，案件被归档。

**A falta de recursos, la investigación se detuvo.**
由于缺乏资源，研究停止了。
