# el criterio

## 📖 基础信息

| 项目 | 内容 |
|------|------|
| **单词** | **el criterio** |
| 音标 | [el kɾiˈte.ɾjo] |
| 词性 | 名词 (m.) |
| 核心释义 | 标准、准则、判断力 |

---

## 🧠 记忆辅助

**联想记忆：**
- 来自希腊语 kriterion（判断的标准）
- 联想：critical（批判的）→ criterio（标准）

**词源补充：**
- 来自希腊语 kriterion，由 krinein（判断）组成
- criterio 字面意思是"判断的依据"，引申为标准、准则

---

## ✏️ 核心用法

### 1. 作主语/宾语（表示标准）
指做决定或判断的依据。

### 2. 与动词搭配
常见搭配：aplicar/ seguir/ establecer un criterio

### 3. 学术/专业领域
指评估或研究的标准。

### 4. 固定短语
- según el criterio de：根据...的标准
- criterio de evaluación：评估标准

---

## 📝 例句模块

### 用法1：标准
**El criterio de selección es la experiencia.**
选择的标准是经验。

**Hay que aplicar criterios objetivos.**
必须应用客观标准。

### 用法2：与动词搭配
**Establecieron criterios claros para la evaluación.**
他们为评估建立了明确的标准。

**Seguimos el criterio del experto.**
我们遵循专家的标准。

### 用法3：学术/专业
**El criterio de evaluación debe ser transparente.**
评估标准应该是透明的。

**Este estudio utiliza criterios científicos.**
这项研究使用科学标准。

### 用法4：固定短语
**Según el criterio del médico, el paciente está mejor.**
根据医生的判断，病人情况好转。

**El criterio de evaluación incluye varios aspectos.**
评估标准包括几个方面。
